# Silentium — 分阶段开发步骤

## 阶段一：基础底座 ⬅ 当前阶段

### 1.1 页面框架
- [ ] 创建 `index.html` 单页应用入口
- [ ] 引入 Tailwind CSS v3 CDN
- [ ] 引入 Font Awesome 6 CDN
- [ ] 引入 Inter 字体
- [ ] 创建 `css/style.css`：CSS 自定义属性（主题变量）、全局样式
- [ ] 实现侧边栏导航布局（240px 宽，移动端滑入式）
- [ ] 实现三个主视图容器：`#view-materials`、`#view-training`、`#view-dashboard`
- [ ] 视图切换路由逻辑（`js/app.js`）
- [ ] 侧边栏导航高亮当前视图

### 1.2 主题系统
- [ ] 创建 `js/theme.js`
- [ ] 实现 CSS 变量驱动的深浅色主题
- [ ] 默认跟随 `prefers-color-scheme`
- [ ] 手动切换按钮（侧边栏底部），偏好存入 LocalStorage
- [ ] 监听系统主题变化事件

### 1.3 数据层
- [ ] 创建 `js/utils.js`：UUID 生成、XSS 过滤、日期格式化、防抖/节流
- [ ] 创建 `js/data-structure.js`：定义 Material、Sentence、TrainingRecord、VocabularyItem、Settings 的默认值和校验函数
- [ ] 创建 `js/storage.js`：
  - LocalStorage 读写封装（`getItem`/`setItem`/`removeItem`，JSON 序列化/反序列化，try-catch 保护）
  - IndexedDB 封装（`openDB`、保存 Blob、读取 Blob、删除 Blob）
  - `initStorage()`：首次运行时初始化默认数据结构
  - 容量检查函数

### 1.4 素材库
- [ ] 创建 `js/materials.js`
- [ ] 新建素材模态框：标题输入、音频文件选择器（File API）、原文文本域
- [ ] 音频文件 → ArrayBuffer → IndexedDB 存储
- [ ] 素材列表渲染（卡片式，含状态标签、句数、时长）
- [ ] 编辑素材（修改标题、替换原文）
- [ ] 删除素材（确认对话框 + 级联删除 IndexedDB 音频 + 训练记录）
- [ ] 空白状态占位图（无素材时引导新建）

### 1.5 音频播放器
- [ ] 创建 `js/player.js`
- [ ] HTMLAudioElement 封装
- [ ] 播放/暂停
- [ ] 进度条（可点击跳转）
- [ ] 时间显示（当前 / 总时长）
- [ ] 倍速选择器（0.5x / 0.75x / 1x / 1.25x / 1.5x）
- [ ] 单句循环播放（接收 startTime, endTime）
- [ ] 音频加载状态指示

### 1.6 JSON 导入导出
- [ ] 导出功能：收集所有数据 → JSON Blob → 触发下载
- [ ] 导入功能：FileReader 读取 → 校验 JSON 结构 → 合并/覆盖确认 → 写入
- [ ] 导出时包含音频文件名映射（实际 Blob 在 IndexedDB）
- [ ] 导入时提示音频文件需单独导入

### 1.7 阶段一验证清单
- [ ] 创建素材后刷新页面，数据仍在
- [ ] 音频播放、暂停、跳转、倍速正常
- [ ] 深浅主题切换正常，刷新后保持
- [ ] 移动端侧边栏滑入/滑出正常
- [ ] JSON 导出文件下载成功
- [ ] JSON 导入数据恢复成功
- [ ] XSS 测试：输入 `<script>alert(1)</script>` 不执行

## 阶段二：核心听写与 AI 分句

### 2.1 文本自动分句
- [ ] 创建 `js/segmentation.js`
- [ ] 实现 `autoSegment(text)`：按 `.` `!` `?` + 换行分割
- [ ] 智能处理缩写（Mr. Dr. etc.）和数字小数点
- [ ] 分句列表 UI：每句显示序号 + 文本预览
- [ ] 手动合并相邻两句
- [ ] 手动拆分一句为两句（点击句中位置拆分）

### 2.2 时间戳匹配
- [ ] 每句显示起止时间输入框（`mm:ss.ms` 格式）
- [ ] 可编辑起止时间
- [ ] 拖拽式时间轴（range input × 2 per sentence）
- [ ] "播放此句"按钮，实际播放音频的 `[startTime, endTime]` 段落
- [ ] 全局播放器在分句模式下联动高亮当前播放句

### 2.3 逐句听写交互
- [ ] 创建 `js/dictation.js`
- [ ] 听写界面布局：上排播放控制，中间隐藏的原文，下排输入框
- [ ] 播放句子音频（可设定重复次数 1-5 次）
- [ ] 用户输入听写文本
- [ ] 提交后自动逐词比对：
  - 正确词：正常显示（绿色或无色）
  - 错误词：红色高亮 + 显示正确词
  - 漏词：下划线占位 + 显示缺失词
  - 多词：灰色删除线
- [ ] 比对算法：LCS（最长公共子序列）或 diff-match-patch
- [ ] 自动进入下一句

### 2.4 3 级提示系统
- [ ] 💡 提示按钮（每句旁边）
- [ ] 第 1 次点击：显示 "This sentence has N words."
- [ ] 第 2 次点击：显示 "W _ _ _ s _ _ _ _ _ _ ..."（首字母 + 下划线）
- [ ] 第 3 次点击：显示 "W _ r d s _ e _ t _ _ c e ..."（首字母 + 元音字母 + 下划线）
- [ ] `hintUsed` 计数不可超过 3
- [ ] 提示状态持久化到 LocalStorage

### 2.5 全文精读面板
- [ ] 听写全部完成后自动解锁（或手动确认解锁）
- [ ] 错词汇总区：列表显示所有错词、漏词、多词
- [ ] 难句标记：每句旁 ⭐ 按钮，标记为"困难句"
- [ ] 笔记区：富文本或纯文本，整篇笔记
- [ ] 单句笔记：展开每句后可添加单句笔记

### 2.6 错题集/重练列表
- [ ] 侧边栏动态生成"错题列表"
- [ ] 显示：句子序号 + 错误词数 + 提示使用次数
- [ ] 按错误数量排序
- [ ] 点击任一项跳转到该句听写
- [ ] "重练全部错题"按钮
- [ ] 重练模式下只出现错题句子

### 2.7 阶段二验证清单
- [ ] 分句结果合理，可手动调整
- [ ] 时间戳拖拽正常，播放段落准确
- [ ] 听写比对结果准确
- [ ] 3 级提示按顺序显示
- [ ] 精读面板解锁逻辑正确
- [ ] 错题重练流程完整

## 阶段四：增值功能（部分）

### 4.1 在线词典
- [ ] 创建 `js/dictionary.js`
- [ ] 单词检测：双击/hover 文本中单词
- [ ] 弹出悬浮窗：音标、词性、释义、例句
- [ ] 主 API（Free Dictionary）请求 + 解析
- [ ] 备用 API（Datamuse）降级
- [ ] 加载/错误状态 UI
- [ ] "加入生词本"按钮

### 4.2 计时专注模式
- [ ] 创建 `js/focus-mode.js`
- [ ] 专注模式入口（导航栏或工具栏按钮）
- [ ] 精简界面：隐藏侧边栏、导航、非必要元素
- [ ] 自定义倒计时（默认 25 分钟，可调 5/15/25/45/60）
- [ ] 倒计时显示（大号数字）
- [ ] Page Visibility API 切屏监听 + 警告 toast
- [ ] 暂停/继续/提前结束
- [ ] 退出时二次确认对话框
- [ ] 完成后记录时长到 LocalStorage

### 4.3 进度看板
- [ ] 创建 `js/dashboard.js`
- [ ] 10 篇目标网格（2×5 或响应式）
  - 未开始：灰底虚线框
  - 进行中：主色边框 + 阶段进度条
  - 已完成：绿色底 + ✓
- [ ] 点击格子跳转到对应素材
- [ ] 单篇进度卡片（弹出或侧面板）：
  - 训练阶段状态
  - 提示依赖率（hintUsed 总计 / 最大可能次数）
  - 错词统计
  - 困难句数量
- [ ] 全局统计区：
  - 总训练句数
  - 平均提示依赖率
  - 已完成篇数 / 10
  - 总训练时长
- [ ] 里程碑 toast（完成第 1/3/5/8/10 篇时提醒）

### 4.4 智能复习提醒
- [ ] 创建 `js/review.js`
- [ ] 看板顶部"待复习提示条"：显示待复习篇数
- [ ] 复习状态自动更新：背诵完成后 → `need_review`，手动标记 → `mastered` / `strengthen`
- [ ] `nextReviewDate` 计算：完成日 + 1 天 / 3 天 / 7 天 / 30 天（按复习次数递增）
- [ ] 抽检复习模式入口
- [ ] 随机抽取 N 句（从已完成的素材中），随机分配听写或跟读
- [ ] 复习完成后更新状态

### 4.5 独立生词本页面
- [ ] 创建 `js/vocabulary.js`
- [ ] 生词本视图（可作为导航项或侧边栏子项）
- [ ] 按素材分类折叠列表
- [ ] 单词卡片：单词、音标、释义、来源句、添加时间
- [ ] hover/点击查词（联动词典）
- [ ] 删除单词
- [ ] "全部导出为 JSON"按钮

### 4.6 阶段四验证清单
- [ ] 词典查询正常，降级逻辑触发正确
- [ ] 专注模式倒计时准确，切屏提醒生效
- [ ] 进度看板数据准确，里程碑触发
- [ ] 复习提醒逻辑正确，抽检可用
- [ ] 生词本增删查正常
