# Silentium — 产品需求文档

## 产品概述

**Silentium**是一个纯前端的英语精听训练工具，严格复刻"尚雯婕十篇精听背诵学习法"，帮助用户通过逐句精听、默写、精读、背诵的完整闭环，深度掌握 10 篇英语素材。

## 目标用户

- 英语中高阶学习者（具备基础词汇量，希望提升听力与语感）
- 崇尚"少即是多"的深度学习理念者
- 需要在浏览器内快速开始训练，无需安装软件

## 核心训练链路

```
选材 → 逐句听写 → 全文精读 → 同速跟读背诵 → 智能复习
```

**不可跳跃**：每步必须完成后才能解锁下一步。

## 功能需求

### 模块一：素材管理
- 新建素材：填写标题、上传音频文件、粘贴英语原文
- 素材列表：查看所有素材及其训练状态
- 编辑/删除已有素材
- JSON 导入/导出（合并或覆盖模式），支持备份与跨设备迁移

### 模块二：训练 — 分句与时间戳
- 文本自动分句（按 `. ! ?` 和换行分割）
- 手动合并/拆分句子
- 手动拖拽调整每句的起止时间戳
- 静音检测辅助分句（可选，后期优化）

### 模块三：训练 — 逐句听写
- 播放句子音频，原文隐藏
- 用户输入听写内容
- 逐词比对：错词标红、漏词下划线、多词灰显
- **3 级提示系统**：
  - 💡 第 1 次：显示单词数量
  - 💡 第 2 次：显示每个单词的首字母
  - 💡 第 3 次：显示每个单词的元音字母
  - 每句记录 `hintUsed`（0-3）

### 模块四：训练 — 全文精读
- 听写全部完成后解锁
- 汇总所有错词
- 手动标记难句
- 笔记区（自由文本）
- 错题集侧边栏：按错误类型分类，支持一键重练

### 模块五：跟读背诵（延后开发）
- 录音功能（MediaRecorder）
- 双轨播放器（原声 / 用户录音）
- 双轨波形图对比
- 语音识别评分（Web Speech API）
- 影子跟读模式
- 背诵打卡

### 模块六：进度看板
- 10 篇目标总览
- 单篇进度卡片（阶段状态、提示依赖率等）
- 全局统计数据
- 里程碑提醒（如"已完成第 5 篇"）

### 模块七：智能复习
- 背诵完成后自动标记复习状态
- 三种状态：已掌握（mastered）/ 需复习（need_review）/ 待强化（strengthen）
- 看板顶部提示待复习篇数
- 抽检复习模式：随机抽句听写或跟读

### 模块八：在线词典
- 点击/hover 单词弹出悬浮窗
- 显示音标、词性、释义
- 主 API：Free Dictionary API，备用：Datamuse API
- API 不可用时优雅降级

### 模块九：生词本
- 独立页面，按素材分类展示
- 联动词典查询
- 支持手动添加/删除

### 模块十：计时专注模式
- 精简界面，隐藏非必要元素
- Page Visibility API 切屏监听
- 自定义专注时长倒计时
- 退出时二次确认
- 计时统计

### 延后开发
- 学习周报卡片（Canvas 绘制）
- 语音录音与评分全流程
- 移动端深度适配

## 非功能需求

### 安全
- 所有用户输入做 XSS 过滤，禁止 `innerHTML` 直接插入用户内容
- JSON 导入必须校验结构，拒绝脚本执行
- 不引入不安全 CDN 资源

### 性能
- 大音频文件加载优化
- LocalStorage 容量监控（超出时提醒用户清理）
- IndexedDB 存储音频 Blob

### 兼容性
- Chrome / Edge / Firefox 最新 3 个版本
- 移动端主流浏览器基本可用

### 数据持久化
- 所有用户操作实时保存
- 刷新页面不丢失数据
- 音频文件通过 IndexedDB 持久化
